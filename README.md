# My Archive (MVP)

## Run locally
```bash
npm install
npm run dev
```

## Add albums
Edit `data/albums.json` and add cover images under `public/covers/...`

## Quick 기록(가장 쉬운 방법)
- Music에서 앨범 클릭
- Album Note / Track Note에 바로 적기 (자동 저장: 브라우저 localStorage)
- Backup에서 JSON 내보내기/가져오기로 백업

## Deploy (Vercel)
- Push to GitHub
- Import in Vercel
- Deploy
